<h3>Homework №18:</h3>
<h3>Wrap the application in a Docker container</h3>

<p>
Deploy the application in a modern DevOps stack.
Result:Application wrapper in Docker.
</p>

<h3>⚙️ Requirements</h3>

<ul>
  <li>Wrap the application in a Docker container. The Dockerfile is typically located at the root of the repository. The image should contain the application's JAR. Building in a container is recommended, but not required.</li>
  <li>There's no need to wrap the database in its own container (unless you're using custom plugins).</li>
  <li>Configure communication between containers using docker-compose.</li>
</ul>
